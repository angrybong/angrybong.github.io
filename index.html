Spring Batch 학습 가이드 (Hugo Draft)

Fast Campus 「실무를 위한 Spring Batch의 모든 것」 강의를 기반으로 한 개인 학습 노트입니다.
모든 글은 draft: true 로 설정되어 Hugo 빌드 시 기본적으로 배포되지 않습니다.

Hugo에 붙이는 방법

  1. 이 폴더의 .md 파일을 Hugo 사이트의 content/ 아래로 복사합니다.

    content/
  study/
    spring-batch/
      _index.md
      01-reader-performance.md
      ...

  2. 로컬에서 draft 포함 미리보기:

    hugo server -D

  3. 공개 블로그로 전환할 때는 front matter에서 draft: false 로 변경하고, 학습 가이드 톤을 공개용 글로 다듬습니다.
